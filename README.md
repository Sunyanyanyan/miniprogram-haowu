# 好物登记小站

一个基于微信小程序云开发的好物登记小站，支持物品登记、浏览、搜索、编辑等功能。

## 功能特性

### 用户功能
- ✅ 物品登记（支持最多3张图片）
- ✅ 首页瀑布流展示
- ✅ 关键词搜索
- ✅ 物品详情查看
- ✅ 信息复制
- ✅ 物品编辑和删除
- ✅ 上架/下架管理
- ✅ 举报功能

### 管理员功能
- ✅ 查看所有物品
- ✅ 强制下架/删除
- ✅ 数据统计

## 技术栈

- **前端**：微信小程序
- **后端**：微信小程序云开发
- **数据库**：云开发数据库
- **存储**：云开发云存储

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/your-username/secondhand-platform.git
cd secondhand-platform
```

### 2. 配置云开发环境

1. 在微信开发者工具中打开项目
2. 点击工具栏的"云开发"按钮
3. 创建或选择云开发环境
4. 复制环境 ID

### 3. 修改配置

在 `app.js` 中修改云开发环境 ID：

```javascript
wx.cloud.init({
  env: 'your-env-id',  // 替换为你的环境 ID
  traceUser: true,
});
```

在 `project.config.json` 中修改 appid：

```json
{
  "appid": "your-appid"  // 替换为你的小程序 appid
}
```

### 4. 创建数据库集合

在云开发控制台创建以下集合：

- `items`：物品信息
- `admins`：管理员信息

**权限设置**：

**items 集合**：
- 读权限：所有用户可读
- 写权限：仅创建者可写

**admins 集合**：
- 读权限：仅创建者可读写
- 写权限：仅创建者可写

### 5. 部署云函数

在微信开发者工具中：

1. 展开 `cloudfunctions` 文件夹
2. 右键点击每个云函数文件夹
3. 选择"上传并部署：云端安装依赖"
4. 等待部署完成

### 6. 编译运行

点击"编译"按钮，即可在模拟器中预览

## 使用说明

### 登记物品

1. 点击底部导航栏的"录入"
2. 填写物品信息（标题、描述、用户唯一标识）
3. 上传图片（最多3张）
4. 点击"录入"按钮

### 管理物品

1. 点击底部导航栏的"我的"
2. 查看已登记的物品
3. 点击"编辑"修改物品信息
4. 点击"展示/隐藏"切换状态
5. 点击"删除"删除物品

### 添加管理员

1. 在小程序中打开调试器
2. 在控制台输入：

```javascript
wx.cloud.callFunction({
  name: 'getOpenid'
}).then(res => {
  console.log(res.result.openid)
})
```

3. 复制输出的 openid
4. 在云开发控制台的 `admins` 集合中添加记录：

```json
{
  "_openid": "你的openid"
}
```

## 注意事项

- 请勿提交隐私信息到 GitHub
- 云开发环境 ID 和小程序 appid 需要替换为自己的
- 图片上传大小限制为 10MB
- 内容安全检测需要开通云开发内容安全服务
